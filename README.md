## Installation

Pour lancer le projet c'est easy :

Installez d'abord Maven, ou si vous utilisez intellIJ c'est déjà intégré

Ensuite,

Installation des dépendances :

```bash
  mvn clean install
```

Lancement du projet :
```bash
  mvn tomcat7:run
```
